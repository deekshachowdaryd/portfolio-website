🤟 ASL Gesture Recognition — Comparative ML/DL Study

An automated American Sign Language (ASL) gesture recognition system that classifies 10 predefined ASL words from dynamic video inputs. This project features a comparative analysis of classical Machine Learning and Deep Learning architectures trained on a custom-recorded video dataset.

Tech Stack

Python PyTorch MediaPipe OpenCV Scikit-learn Pandas NumPy Matplotlib Seaborn

Model Comparisons & Performance

Four models were developed, trained, and evaluated using k-fold cross-validation on accuracy, precision, recall, and F1-score:

  • 🥇 1D CNN (97.0% Accuracy / 0.97 F1-Score): Best performing model. Treats the hand landmark sequence as a 1D signal. Captures local spatial shape patterns extremely well across gesture frames.
  • 🥈 CNN + LSTM Hybrid (95.0% Accuracy / 0.95 F1-Score): Linear feature extractor compresses frame landmarks into 128-d vectors, which are then classified by an LSTM.
  • 🥉 BiLSTM + Attention (89.9% Accuracy / 0.90 F1-Score): Processes sequential frames bidirectionally, applying attention weights to highlight critical keyframes.
  • 4th RandomForest (88.1% Accuracy / 0.965 F1-Score): Traditional ML baseline. Achieved high precision but lower overall accuracy due to a lack of temporal modeling.

Key Finding: The 1D CNN outperformed all sequential models because ASL gestures in this dataset are differentiated more by the spatial hand shapes than by complex motion patterns over time.

Key Features

  • Custom Video Dataset: Built a dataset of 100 videos (10 per class) covering 10 distinct ASL words (Airplane, Bad, Child, Drink, Drop, Sit, Stop, Teacher, Why, Yes) under controlled environment lighting and camera angles.
  • Landmark Preprocessing: Uses MediaPipe Hand Landmarker to extract 21 3D coordinates (126 total values per frame for both hands), avoiding raw pixel processing to keep models highly lightweight and robust to lighting/background variation.
  • Exploratory Data Analysis (EDA): Diagnostic scripts to check video characteristics (duration, frame brightness, wrist movements) and generate comparative plots.

Challenges

Designing a robust pipeline that normalizes hand coordinates regardless of the user's distance from the camera or their hand size. Managing temporal sequence padding for variable-length video files while preventing neural network overfitting on a custom dataset required careful dropout tuning (0.3 - 0.4) and Xavier weight initialization.

What I Learned

Deepened my expertise in comparing machine learning baselines to deep convolutional and recurrent architectures in PyTorch. Gained practical experience in computer vision pipeline design by extracting spatial landmarks with MediaPipe to simplify down-stream deep learning models.
