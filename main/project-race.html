F1 Race Outcome Predictor

A machine learning project that predicts Formula 1 race outcomes based on historical race data, driver performance, and team statistics.

Tech Stack

Python Pandas NumPy Scikit-learn FastF1

Key Features

  • Data collection using FastF1 API
  • Feature engineering using driver and constructor stats
  • Gradient Boosting model for prediction
  • Evaluation using MAE (Mean Absolute Error)

Challenges

Cleaning and structuring real-world F1 data was complex. Handling inconsistencies and selecting meaningful features for prediction required experimentation.

What I Learned

Gained hands-on experience in machine learning workflows, including preprocessing, model training, and evaluation. Also improved my data analysis skills.
